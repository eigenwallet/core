eigenwallet Flatpak Repository

This repository provides Flatpak packages for eigenwallet GUI.

Installation

# Add the repository
flatpak remote-add --if-not-exists eigenwallet https://eigenwallet.github.io/core/eigenwallet.flatpakrepo

# Install
flatpak install eigenwallet org.eigenwallet.app

# Run
flatpak run org.eigenwallet.app

Run it

flatpak run org.eigenwallet.app

Update

flatpak update org.eigenwallet.app

Requirements

  • Flatpak 1.0 or newer
  • Linux distribution with Flatpak support
  • x86_64 architecture
